## Run Locally

Clone the project

```bash
  git clone git@github.com:anthonyardhuin/minishell.git
```

Go to the project directory

```bash
  cd minishell
```

Compile the project

```bash
  make
```

Start the terminal

```bash
  ./my_sh
```
